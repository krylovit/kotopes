# Нейросеть-трейдер Pro

LSTM (Long Short-Term Memory) нейросеть для прогнозирования финансовых рынков на основе данных Binance API.

## Структура проекта

```
├── index.html          # Главная страница приложения
├── css/
│   └── styles.css      # Стили интерфейса
└── js/
    ├── config.js       # Конфигурация (API, параметры модели)
    ├── state.js        # Глобальное состояние приложения
    ├── utils.js        # Утилиты (уведомления, логи, загрузчик)
    ├── indicators.js   # Технические индикаторы (RSI, SMA, расчеты)
    ├── model.js        # LSTM модель (создание, обучение, сохранение)
    ├── data.js         # Работа с Binance API, обработка данных
    ├── charts.js       # Графики (Chart.js)
    ├── ui.js           # Обновление интерфейса
    └── app.js          # Главный цикл, обработчики событий, инициализация
```

## Технологии

- TensorFlow.js (LSTM нейросеть)
- Chart.js (визуализация)
- Binance API (данные о криптовалютах)
